Proyectos

Shopys

Java Spring Boot -- Angular

Shopys es un sistema de agendamiento de citas desarrollado con Java Spring Boot y Angular, que facilita la programación de citas y envía notificaciones por correo automáticamente. Combina la robustez de Spring Boot con la interactividad de Angular para ofrecer una experiencia de usuario fluida y eficiente.

Persistence

Java Spring Boot -- Angular

desarrollado en Java nativo, que utiliza archivos de texto para guardar, leer y editar datos. Este enfoque simplificado asegura una gestión eficaz de las citas sin la complejidad de bases de datos avanzadas.

PruebaTech

Vue Js

PruebaTech es una aplicación desarrollada en Vue.js diseñada para consumir servicios API REST. Esta herramienta facilita la interacción con diversas APIs, permitiendo a los usuarios realizar solicitudes y recibir respuestas de manera eficiente. PruebaTech aprovecha las capacidades reactivas de Vue.js para ofrecer una experiencia de usuario fluida y dinámica, ideal para probar y depurar servicios web RESTful.
